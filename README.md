Library Management System using SQL

Project Overview

This project is a Library Management System developed using SQL. It helps manage books, members, book issuing, and return records efficiently.

Features

- Add and manage books
- Manage library members
- Issue books to members
- Return books
- Track available books
- Generate library records

Technologies Used

- SQL
- MySQL

Database Tables

- Books
- Members
- Issue Records
- Return Records

Skills Demonstrated

- Database Design
- SQL Queries
- Data Management
- Relational Database Concepts
